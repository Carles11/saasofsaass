# Feature: AI-Translation Engine
- **Goal:** Automate content localization into 8 locales (en, es, ca, eu, ga, fr, it, de).
- **Provider:** Gemini 2.5 (via server action).
- **Trigger:** User clicks "Add Language" in SiteBuilder → Gemini translates all blocks and entities for that locale.
- **Storage:** Block translations saved to `translations` JSONB column on `blocks` table; entity translations saved to `tenant_translations` table.
- **Rules:**
  - Tenant detects `defaultLocale` — source language for translation.
  - Never overwrite existing translations — only fill empty/missing locale keys.
  - Platform UI strings stored in `platform_translations` table (namespace/key/locale). Unique on `(namespace, key, locale)`.
