# Gemini/DeepL caching & 11-lang logic.
# Feature: AI-Translation Engine
- **Goal:** Automate content localization into 11 languages.
- **Provider:** Gemini 3 Flash (via Supabase Edge Functions).
- **Trigger:** Database Webhook on `content_blocks` (INSERT/UPDATE).
- **Logic:**
  1. Detect `default_lang` of tenant.
  2. Send `content` JSONB to Gemini.
  3. Map response to the `translations` table.
- **Rules:**
  - Never overwrite manual translations if a "lock" flag is present.
  - Maintain HSL brand variables even in translated strings.
