# Feature: SEO & GEO (Generative Engine Optimization)
- **Goal:** Maximize extractability for both Google and LLMs (Gemini/Perplexity).
- **Core Components:**
  - **Dynamic Metadata:** `generateMetadata` must use tenant data to set per-locale titles, OG, Twitter, hreflang, canonical.
  - **JSON-LD Engine:** Generate Schema.org (PodcastEpisode, NGO, Article) dynamically.
  - **Hreflang:** Automate for the 8-locale matrix.
- **AI-Search Rules:**
  - Use semantic HTML tags (`<article>`, `<section>`).
  - Generate an "AI Cheat Sheet" (hidden metadata) in the footer describing the Tenant Entity.
