# Feature: Success-Based Paywall (Future)
- **Goal:** Enable tenants to monetize blocks while SoSS takes 1%.
- **Provider:** Stripe Connect (Express/Standard).
- **Architecture:**
  - Wrap paywalled content behind access level check.
  - Logic: If `block.access_level === 'paid'`, check `user_subscriptions`.
- **Payment Flow:**
  - `application_fee_amount` = `total * 0.01`.
  - Destination: `tenant.stripe_connect_id`.
- **Rules:**
  - Always handle Stripe Webhooks for "Subscription Cancelled" events.
  - UI must show a "Locked" state using shadcn/ui skeleton/blur.
