# Entity: Tenant & Domain Management
- **Goal:** Resolve `subdomain.localhost:3000` (dev) or `*.saasofsaass.com` (prod) to a `tenant_id`.
- **Mechanism:** `src/proxy.ts` middleware reads hostname → looks up `domain` or `slug` in tenants table via Drizzle → rewrites to `/(tenants)/[domain]`.
- **Templates:** Tenant has `templateId` text field (default `"default"`). Runtime resolves via `TEMPLATES[tenant.templateId] || TEMPLATES.default` — appearance-only presets (themeClass, containerClass).
- **Branding:** JSONB column on tenants — HSL vars, logo, fonts. Injected via CSS variables in tenant layout.
