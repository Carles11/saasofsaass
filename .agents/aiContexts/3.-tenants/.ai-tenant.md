# Entity: Tenant & Domain Management
- **Goal:** Resolve `subdomain.soss.com` or `customdomain.com` to a `tenant_id`.
- **Mechanism:** Next.js Middleware + Supabase query.
- **DNS States:** 1. `DNS_REQUIRED`: Tenant needs to set A/CNAME records.
  2. `SSL_GENERATING`: Vercel is issuing cert.
  3. `LIVE`: Site is active.
  4. `ERROR`: Configuration mismatch.
- **Theming:** Inject `primary_color` (HSL) into `:root` CSS variables via the layout.
